[업로드 파일 내용]

- 회원가입 기능 구현(미완성)
회원 가입페이지와 회원가입 실패 페이지, 성공페이지를 만들어 
해당하는서블릿 파일로 DB에 해당 아이디+닉네임이 있는지 조회해서
있으면 false 회원가입 실패 처리, 없으면 DB에 값을 넣어 등록후 
회원가입 성공 페이지로 이동하는 기능 구현 중

[발생 문제]
-아이디, 닉네임 중복 확인은 jsp페이지를 이용하면 불필요한 동작을 요구하게됨
해당 문제는 프론트의 javascript를 이용해 중복 확인 버튼을 누르면 구현해 놓은
서블릿에 요청해서 DB에 접속해 중복을 체크한 후 레이블로 처리하는 과정이 필요
프론트와 백엔드 파트의 상의가 필요

-jsp usebean을 사용하여 회원가입 요소들을  DTO에 담아 넘기는,
즉, jsp페이지 html태그들의 요소값을 useBean을 사용해 서블릿으로 넘기는 방법 조사 필요
