프로젝트의 시연 순서입니다.

시연에서는 사용자A와 사용자B로 구분하여, 회원가입을 하고, 로그인을 하며,
학교 소개 기능을 이용 후, 한명의 사용자가 멘토링 게시물을
작성하고, 다른 사용자는 해당 멘토링을 예약 후, 메타버스를 다운로드하여
클라이언트를 설치 후, 각 사용자 별로 캐릭터를 생성하여 가상 공간에서 사용하는
다양한 기능을 시연하겠습니다. 

시연에 앞서 간단하게 Men-Meet의 사용방법을 발표하겠습니다.

회원가입은 통상적인 서비스의 회원가입처럼, 각 입력 필드에 입력하시고, 회원가입 버튼을 누르면 됩니다.
로그인도 마찬가지로 아이디와 비밀번호를 입력하고 로그인 버튼을 통해 로그인할 수 있습니다.
마이페이지의 이름 변경과 비밀번호 변경인데, 이름 변경은 간단하게 바꿀 이름을 입력하시면 되고, 비밀번호 변경은 현재 비밀번호와 새 비밀번호, 그리고 새 비밀번호를 한번 더 입력하시고 버튼을 클릭하시면, 비밀번호가 변경됩니다.
예약 취소도 다음 사진과 같이 버튼 한번만 누르시면 됩니다.
멘토링을 이용하시고 싶다면, 멘토와 멘티 중 하나를 선택하셔서 제공되는 리스트를 확잍하시면 됩니다. 해당 게시글을 클릭하면, 상세한 화면이 나오며, 아래에 있는 멘토/멘티 신청하기 버튼을 통해 멘토링을 예약 할 수 있습니다.
만약 나와있는 멘토링이 아닌 새로운 멘토링을 하고 싶다면, 글쓰기 버튼을 통해 이용하실 수 있습니다. 멘토링을 받고 싶은 대상을 지정해 주고, 분야를 선택하고, 하고싶은 멘토링의 제목과 설명을 적어주시고, 일정을 선택해주시면 됩니다!
메타버스 소개 다운로드는 간단한 버튼 하나를 클릭해주시면 됩니다.
마지막으로 학교 소개를 이용하시고 싶다면 SEOIL 탭을 클릭하고, 원하는 카테고리의 버튼을 클릭해 주시면 됩니다!
